To run the code I have submitted in the "Big Data Project 1" file, follow the instructions below.

Firstly, use this command to make the code executable by your user: chmod u+x ./example_code.sh

Then to run the code use the following command: ./example_code.sh

To view the code and/or edit the code use the command: nano example_code.sh
